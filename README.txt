# بوابة برامج גפ״ן — استضافة سريعة

**المحتويات:**
- `index.html` — صفحة رئيسية RTL بالعربية
- `docs/` — ملفات الـ PDF الثلاثة

**رفع على GitHub Pages:**
1) أنشئ مستودعًا جديدًا على GitHub و ارفع إليه كل محتويات هذا المجلد.
2) من Settings → Pages → اختر Source: `Deploy from a branch`، و Branch: `main`، Folder: `/root`.
3) خلال دقائق ستحصل على رابط موقعك.

**رفع على Netlify:**
- افتح https://app.netlify.com/ ثم اسحب المجلد بالكامل إلى منطقة "Drag & Drop".

**رفع على Vercel:**
- افتح https://vercel.com/new ثم اختر "New Project" وارفع المجلد.

**رفع على Neocities:**
- أنشئ حسابًا في https://neocities.org/ ثم ارفع المجلد كما هو.

**تعديل الروابط/النصوص:**
- غيّر عناوين الكروت أو النصوص داخل `index.html` حسب الحاجة.
- يمكنك إضافة زر واتساب أو بريد لاحقًا إن رغبت.
